Pryo oppgaver

A: Småtteri

  1. Slutte å dele skolearbeidmappen din på drive.google.com

B: ESD

  1. ESD: https://www.youtube.com/watch?v=qOK7X4ko3YU
  2. Montere ekstra harddisk i pci https://www.youtube.com/watch?v=eKUF1-DEJCE - Ta et par bilder under arbeidet til loggen

C: Sky og passord

  1. Sette opp icloud på windows pc https://support.apple.com/no-no/HT204025
  2. Laste ned og installere keepass https://keepassxc.org
  3. Få keepass på pc til å lese samme fila som telefonen på icloud/Strongbox/

D: GIT

  1. Lage konto på github.com og lage/notere passordet i keepass (det er ett av stegene i den neste oppgaven så du kan begynne på den)
  2. Intro, installasjon og enkel bruk:https://www.youtube.com/watch?v=SWYqp7iY_Tc
  3. Andre videoer: Installere git https://medium.com/devops-with-valentine/2021-how-to-install-git-on-windows-10-step-by-step-guide-1c9db500e734
  4. Litt annen innfallsvinkel https://medium.com/free-code-camp/a-beginners-guide-to-git-how-to-create-your-first-github-project-c3ff53f56861
  5. Legg inn ssh nøkkel for å gjøre kloning og push enklere https://www.youtube.com/watch?v=WgZIv5HI44o

E: Lage webside på github med logg over pryo oppgavene

  1. Lage en webside på github som lister opp det du gjør på pryo https://pages.github.com https://docs.github.com/en/pages/quickstart
  2. Clone websiden din fra github til git på din egen maskin
  3. Begynn på en logg index.html (https://www.w3schools.com/html/html_editors.asp) der hvor du klonet websiden din fra github, add den og commit og push den slik at den kommer ut på nettet
  4. Legg til et bilde fra harddisk monteringa, som kanskje er enkelt syncet til maskinen din hvis du valgte det i icloud setup https://www.w3schools.com/html/html_images.asp

F: Installere opensuse 15.3 på den gamle alienware maskinen, klon websiden din og gjør en oppdatering

  1. Last ned opensuse (ikke ubuntu) på en minnepinne https://www.youtube.com/watch?v=pwWfJwlZLWg
  2. og installer MED KDE på alienware maskinen https://www.youtube.com/watch?v=zJeMqUUhV4U
  3. Klon websiden din, husk du må lage ny ssh nøkkel her og legge inn på github hvis du ikke vil bruke https og skrive passord
  4. Editer webside din med KATE, som er installert hvis du valgte KDE og ikke xfc under installeringa
  5. Push ut endringene
